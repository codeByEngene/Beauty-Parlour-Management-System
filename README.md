#Beauty Parlour Management System

## Setup Instructions 
1. Clone repository
2. Import `database/pms_db.sql` into phpMyAdmin
3. Configure database in `config.php`
4. Run project on XAMPP localhost

## Features
- User/Admin login system
- Appointment booking
- Service management
- Contact system
- Invoice generation
